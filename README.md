Project name:Guess the word game 
Description:This C++ program is a simple “Guess the Word” game. A random word is selected from a list, and the player guesses letters to reveal the hidden word. The game checks each input, updates correct guesses, and decreases attempts for wrong guesses. It demonstrates loops, conditions, strings, vectors, and random number generation in C++.
